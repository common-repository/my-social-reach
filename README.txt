== Description ==

Activate the plugin, and it will automatically add social sharing buttons at the end of the post content automatically. 

This plugin enables your website users to share your posts over Facebook, Twitter, Telegram, Linkedin, Whatsapp, Pinterest & Email.

This is the Simplest and easiest Social Sharing plugin with share count for Facebook, Twitter, Telegram, Linkedin, Whatsapp, Pinterest & Email 

== Installation ==

Upload the extracted plugin files to the /wp-content/plugins directory, or install the plugin through the WordPress plugins screen directly.

Activate the plugin through the "Plugins" menu in WordPress.
